1.1 · Estandares y unidades

Inicio Ejemplo

Idea central

La fisica necesita mediciones comparables. Para eso se usan estandares y un sistema comun de unidades que permita expresar longitudes, masas, tiempos y otras magnitudes sin ambiguedad.

  • Un estandar define de manera reproducible como medir una magnitud.
  • Una unidad es la referencia concreta con la que comparamos una medicion.
  • El Sistema Internacional (SI) organiza las unidades base y sus prefijos.

En la practica, dominar prefijos como mili, micro, kilo o mega evita errores de conversion y de orden de magnitud.

Unidades base frecuentes

  • Longitud: metro (m)
  • Masa: kilogramo (kg)
  • Tiempo: segundo (s)
  • Temperatura: kelvin (K)
  • Corriente electrica: ampere (A)

A partir de estas unidades se construyen otras magnitudes derivadas, por ejemplo velocidad \((\text{m/s})\), aceleracion \((\text{m/s}^2)\) o fuerza \((\text{kg·m/s}^2)\).

Prefijos metricos utiles

Los prefijos SI permiten escribir cantidades muy grandes o muy pequenas sin llenar el resultado de ceros. Por ejemplo, \(1\,\text{km} = 10^3\,\text{m}\) y \(1\,\text{mm} = 10^{-3}\,\text{m}\).

Chequeos rapidos

Ejemplo 1 · Conversion sencilla

\(3.2\,\text{km} = 3.2 \times 10^3\,\text{m} = 3200\,\text{m}\).

Ejemplo 2 · Escala pequena

\(250\,\mu\text{m} = 250 \times 10^{-6}\,\text{m} = 2.5 \times 10^{-4}\,\text{m}\).

Ejemplo 3 · Orden de magnitud

Un error comun es confundir mili \((10^{-3})\) con micro \((10^{-6})\). La diferencia es un factor de \(10^3\).
